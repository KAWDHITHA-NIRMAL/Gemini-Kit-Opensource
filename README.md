## 🌟 Overview

**Gemini-Kit Premium** is a high-end, unofficial open-source web interface for Google Gemini. Built with a focus on **Glassmorphism (White & Blur)** aesthetics and seamless user experience, this application leverages the powerful `@k.nirmal/gemini-kit` wrapper to provide a professional AI chat environment without the need for an official API key.

Designed by **CodeX Developers**, it brings the full power of Gemini Vision, Text-to-Speech, and real-time status monitoring into a sleek, responsive dashboard.

---

## 🚀 Key Features

### 🎨 Next-Gen Design
- **White & Blur Glassmorphism**: A stunning, premium aesthetic using Tailwind CSS and backdrop-blur filters.
- **Dynamic Sidebar**: Manage your workflow with a professional sidebar featuring "New Chat" actions and activity history.
- **Micro-Animations**: Smooth message entry and state transitions powered by custom CSS keyframes.

### 🧠 Advanced AI Capabilities
- **Multi-Turn Chat**: Intelligent conversation memory (Context-aware).
- **Gemini Vision**: High-speed image uploads for visual analysis and OCR.
- **Instant TTS**: Listen to Gemini's responses with built-in Text-to-Speech orchestration.
- **Rich Formatting**: Full Markdown support (Headings, Tables, Lists) and Syntax Highlighting (for 100+ languages).

### ⚡ Professional Infrastructure
- **Real-time Status Badge**: Monitors both internet connectivity and server health with visual "Ping" indicators.
- **Image Preview System**: Elegant upload panels with "Safety Lock" (pre-upload validation).
- **One-Click Share**: Built-in logic for sharing conversations (supported by underlying kit).

## 🛠️ Stack & Technologies

- **Frontend**: HTML5, Vanilla JavaScript, [Tailwind CSS](https://tailwindcss.com/)
- **Backend**: [Node.js](https://nodejs.org/), [Express.js](https://expressjs.com/)
- **Core Library**: [@k.nirmal/gemini-kit](https://www.npmjs.com/package/@k.nirmal/gemini-kit)
- **Formatting**: Marked.js (Markdown), Highlight.js (Code highlighting)
- **Fonts**: Outfit (Brand), Inter (Reading)

## 📦 Installation & Setup

1. **Clone the Repository**
   ```bash
   git clone https://github.com/KAWDHITHA-NIRMAL/Gemini-Kit-Web.git
   cd Gemini-Kit-Web
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Cookies**
   Create a `cokies.json` file in the root directory and paste your Gemini cookies (JSON array format).

4. **Run the App**
   ```bash
   npm run start
   ```
   *Access the app at `http://localhost:3000`*
